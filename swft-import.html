Import Customers

Upload a CSV file exported from Jobber, ServiceTitan, or any spreadsheet.
Jobber: Clients → Export → CSV  |  ServiceTitan: Customers → Export
Expected columns (any order):
  • Name or First Name + Last Name — required
  • Email, Phone, Address, Notes — optional
First row should be column headers. Up to 500 customers per import.

Import Jobs

Upload a CSV file with your job history.
Jobber: Jobs → Export → CSV  |  Customer names are matched automatically.
Expected columns (any order):
  • Title or Name — required
  • Client / Customer Name, Status, Date, Address, Service, Description, Cost — optional
Customers are matched by name to your existing SWFT customers. Up to 500 jobs per import.
